Roadmap do Projeto Crush4Pets

Progresso de Desenvolvimento & Planejamento

Data Atual: 7 de Abril de 2025

Visão Geral do Projeto

Este documento apresenta o roadmap de desenvolvimento para o ecossistema Crush4Pets, que consiste em múltiplos serviços integrados trabalhando juntos para fornecer uma solução abrangente de gestão para pets.

O desenvolvimento está estruturado em quatro sprints semanais, com revisões regulares e sessões de planejamento para garantir a entrega pontual de funcionalidades e correções de bugs.

Cronograma de Sprints

Sprint 1: Configuração Inicial

31 de Março - 7 de Abril de 2025

Início: 31 de Março de 2025 | Revisão: 7 de Abril de 2025 (Hoje)

  • Criar uma empresa no GitHub, chamada crush-4-pets
  • Enviar todos os 4 projetos para o GitHub
  • Criar um projeto orquestrado para automatizar o processo de inicialização local
  • Identificar todas as arquiteturas publicadas no Digital Ocean
  • Implementar Digital Ocean no projeto de upload, removendo os buckets AWS S3
  • Identificar local de publicação ausente para sistema.crush4pets.com.br
  • Correção de vários bugs encontrados no projeto do ERP

Sprint 2: CI/CD & Correção de Bugs

8 de Abril - 14 de Abril de 2025

Planejamento: 8 de Abril de 2025 | Revisão: 14 de Abril de 2025

  • Implementar pipeline de CI/CD no GitHub usando Actions
  • Corrigir todos os bugs identificados pelo cliente
  • Corrigir os problemas de visualização nos mapas
  • Garantir que o alerta de atualização de software seja exibido ao entrar no aplicativo
  • Assegurar o funcionamento correto da localização de georreferenciamento no dispositivo móvel
  • Reposicionar o botão "Excluir Conta" na tela de perfil para aumentar sua visibilidade
  • Implementar todas as funcionalidades solicitadas pelo cliente
  • Implementar a Rede de Distribuição de Conteúdo (CDN) na Digital Ocean
  • Atualizar os endereços de armazenamento e recuperação de imagens em todos os projetos
  • Redirecionar as notificações de novas mensagens para o ícone "Mais" na tela principal
  • Configurar as notificações para aparecerem apenas com mensagens novas não lidas
  • Revisar e melhorar o processo de atualização de informações para perfis de pets
  • Implementar a funcionalidade de exclusão de conta ao acionar o botão correspondente
  • Permitir que o usuário clique no telefone de profissionais/anúncios e inicie a ligação
  • Garantir que todos os projetos estejam funcionando corretamente

Sprint 3: Submissão para as Lojas

15 de Abril - 21 de Abril de 2025

Planejamento: 15 de Abril de 2025 | Revisão: 21 de Abril de 2025

  • Submeter todas as correções de bugs para as lojas
  • Submeter todas as funcionalidades implementadas para as lojas
  • Preparar documentação para as submissões nas lojas
  • Conduzir testes pré-submissão

Sprint 4: Implementação de Feedback das Lojas

22 de Abril - 28 de Abril de 2025

Planejamento: 22 de Abril de 2025 | Revisão: 28 de Abril de 2025

  • Resolver todos os feedbacks da revisão das lojas
  • Corrigir todas as solicitações retornadas pelas lojas
  • Reenviar aplicativos atualizados
  • Teste final de garantia de qualidade
